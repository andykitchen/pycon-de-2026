A Structural Causal Model (SCM) is a DAG where each node value is a function of its parents and a exogenous noise term $U_i$

\[ X_i = f_i(X_{pa(i)}, U_i) \]

Towards a methodology

  2. Write down a Bayesian hierarchical model linking parameters to observations
  3. Simulate outcome distribution with/without intervention
  4. Calculate intervention detectability

  4. Calculate the expected value of the intervention given perfect information

  6. Use domain knowledge to set safety margins
